# Инструкция по установке и запуску

## Требования

- macOS с установленным Xcode 14.2 или новее
- iOS 14.2+ симулятор или устройство
- Swift 5.5+

## Установка

1. Откройте проект в Xcode:
```bash
cd "/Users/ss/Desktop/Git/RealEstateAnalyzer-iOS"
open RealEstateAnalyzer.xcodeproj
```

2. Выберите симулятор или устройство для запуска

3. Нажмите ⌘R для запуска приложения

## Структура проекта

Проект организован следующим образом:

- **App/** - Точка входа приложения
- **Models/** - Модели данных (Property, Tenant, Analytics)
- **Services/** - Бизнес-логика (DataManager, MetricsCalculator)
- **Views/** - SwiftUI представления
- **Utilities/** - Вспомогательные расширения

## Миграция данных

Для переноса данных из Python версии:

1. Скопируйте файл `data.json` из Python проекта в Documents директорию приложения
2. При первом запуске приложение автоматически загрузит данные

## Известные ограничения
